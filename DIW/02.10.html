2. Canibalismo SEO

¿Qué es el Canibalismo SEO?

Ocurre cuando varias páginas compiten por las mismas palabras clave.

Solución: Usar la etiqueta <link rel="canonical">



3. Páginas de Meta Tags

Title Description Viewport Charset

Meta Title

Uso: <title>Mi Página Web</title>

Es el título que aparece en Google.

Meta Description

Uso: <meta name="description" content="Texto descriptivo">

Aparece debajo del título en los resultados.

Meta Viewport

Uso: <meta name="viewport" content="width=device-width, initial-scale=1.0">

Hace la web responsive en móviles.

Meta Charset

Uso: <meta charset="UTF-8">

Define la codificación de caracteres.
